## Как финансист использует карточки

### Для расчёта actual_margin по позиции:
```
actual_margin = (продажная_цена - actual_cost) / продажная_цена

actual_cost = Σ(ингредиент × закупочная_цена_из_накладных)
```

### Пример: Капучино М
- Продажная цена: ~200 ₽ (из каталога)
- Себестоимость из карточки: 56 ₽ (базовая, 2025)
- Actual margin: (200 - 56) / 200 = **72%**
- После пересчёта с Флоралом Apr2026: (~60₽) → **70%**

### Для catalog_margin (текущий метод):
Берётся из Продажи.xlsx (Saby) — колонка «Себест.» × количество.

## Что нужно сделать (РП: Пересчёт себестоимости)

1. Открыть Google Sheets «Себестоимость напитков» — обновить цены зерна/молока/сиропов
2. Открыть «Калькуляции КУХНЯ» → лист «Продукты (июнь 2025)» — обновить цены ингредиентов
3. Экспортировать результаты → обновить BAR.COSTING.001 и KITCHEN.COSTING.001
4. Финансист автоматически получит актуальные цены в следующем цикле

**Приоритет:** Напитки важнее кухни (доля выручки ~75% бар vs 25% кухня).
