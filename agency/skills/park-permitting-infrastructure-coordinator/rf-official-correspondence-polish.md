# Skill: RF Official Correspondence Polish

## Зачем нужен

Этот skill нужен, чтобы Park-agent не останавливался на “хорошем черновике”,
а доводил официальное письмо до реально готового документа по стандарту
деловой переписки РФ.

## Когда запускать

- адресат уже выбран;
- предмет письма уже определён;
- документ надо довести до `clean-ready / print-ready / ready-to-send`.

## Что проверить обязательно

### 1. Шапка и адресный блок

- шапка расположена по правому краю;
- адресат указан официально и без сокращений, где это важно;
- отправитель указан полно.

### 2. Реквизиты

- нет пустых `[ФИО]`, `[адрес]`, `[телефон]`, `[email]`;
- реквизиты совпадают с каноническим источником;
- дата и подпись не забыты.

### 3. Предмет обращения

- предмет узкий и проверяемый;
- вопросы пронумерованы;
- ясно, какой именно ответ ожидается.

### 4. Чистота документа

- нет черновых пометок;
- нет техничекских хвостов;
- приложения оформлены;
- документ выглядит готовым к печати и подаче.

## Жёсткие запреты

- Не оставлять плейсхолдеры.
- Не называть документ финальным, если он ещё не вычитан по шапке и реквизитам.
- Не подменять этим skill-ом юридическое заключение.

## Truthful output

После skill документ должен иметь один из статусов:

- `clean-ready`
- `print-ready`
- `ready-to-send`

Если есть хотя бы один незаполненный реквизит, это не выше `draft`.
